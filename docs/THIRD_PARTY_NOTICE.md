# Third-Party Notice

This repository contains a migrated runtime engine used to reproduce the GBSD
experiments. Parts of that runtime engine originate from the earlier Psi-NN /
Bayesian-Psi-NN development code lineage, especially low-level PINN training,
visualization, and structure-reconstruction utilities.

The corresponding publication is:

Liu, Z., Liu, Y., Yan, X. et al. Automatic network structure discovery of
physics informed neural networks via knowledge distillation. Nature
Communications 16, 9558 (2025). https://doi.org/10.1038/s41467-025-64624-3

This notice is included for code provenance and academic attribution. It should
not be read as saying that the GBSD method is identical to Psi-NN. The GBSD
contributions in this repository are the guarded Bayesian structured
distillation workflow, the guard-validation / blind-test protocol, the
uncertainty calibration diagnostics, the unified result contract, and the paper
table consistency checks.
